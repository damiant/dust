# Dust - Product Overview

**Dust** is an offline-first mobile guide for Burning Man events, theme camps, and art installations.

## Core Purpose
Provide festival attendees with comprehensive, searchable information about events, theme camps, and art that works completely offline on iOS and Android devices.

## Key Features
- Event discovery by day, category, theme camp, or search
- Art and theme camp browsing and search
- Location mapping for art and camps
- Favorites system for events, art, and camps
- Geolocation-based nearby discovery
- Event reminders and notifications
- Rock Star Librarian DJ set listings
- Full offline functionality

## Target Users
Burning Man festival attendees who need reliable access to event information without cellular connectivity.

## Deployment
- iOS: App Store
- Android: Google Play
- Web: https://dust.events
- Netlify deployment for preview/production builds
